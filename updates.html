Update Log

Track the evolution of ControlBans. We focus on stability, performance, and cross-server consistency.

v4.3 The Stability Update

January 2026
LATEST

This update shifts focus entirely to performance and reliability. We have removed experimental features to ensure zero-latency punishments on large networks.

REMOVED AI Moderation: Removed Whisper/OpenAI integration. API dependencies caused unnecessary latency and setup complexity for server owners.
PERF Database Optimization: Switched to HikariCP connection pooling optimization for MySQL and MariaDB.
NEW Void Jail Logic: Improved teleportation handling to prevent "falling out of the world" loops on high-latency connections.
FIX Fixed an issue where cross-server mute synchronization would delay by 1-2 seconds on BungeeCord.

v4.2 Experimental

Deprecated
