Current Release

v4.7 — The "Silent Killer" Update

MARCH 6, 2026

After analyzing performance data from high-load networks (50k+ players), we identified and neutralized several "silent killers"—issues that don't crash a server immediately but strangle performance over time. This update makes ControlBans significantly more scalable.

CRITICAL
Import Throttling

Rewrote the ImportService. Importing massive databases (Essentials/LiteBans) no longer floods the thread pool. The system now intelligently throttles file reads, preventing OOM (Out of Memory) crashes during migration.

PERF
Async SkinBan I/O

SkinBan file operations are now 100% asynchronous. Large ban lists will no longer freeze the main thread during saves or loads.

REMOVED
Web Panel

The internal web server caused stability issues on some network configurations. It has been removed. We have shifted focus to improving the Voice Chat Muting integration instead.

DB
History Indexing

Fixed a major SQL bottleneck by adding missing indexes to the History table. Tab-completing commands (`/ban <tab>`) no longer triggers a full table scan, preventing database lockups under load.

FIX
Memory Leak & Soft-Depends

Fixed a leak in ChatInputListener where disconnected players remained in memory. Hardened VoiceChat integration to prevent crashes on servers with outdated API versions.
