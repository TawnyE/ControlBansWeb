Installation

  1. Download the Plugin

    Get the latest jar file from BuiltByBit (Premium) or Modrinth (Free).

  2. Install dependencies

    ControlBans works standalone, but we recommend DiscordSRV (for logs) and SimpleVoiceChat (for AI voice moderation).

  3. Drag & Drop

    Place the `.jar` file into your server's /plugins folder.

  4. Restart

    Restart your server to generate configuration files.

Running a Network?

If using Velocity or BungeeCord, ensure you install the ControlBans-Bridge jar in your proxy's plugins folder for cross-server synchronization.

Migration

Moving from another plugin? We make it easy. Ensure both plugins are installed, then run:

/controlbans import

Supports: LiteBans, Essentials, AdvancedBan.

Punishment Commands

All punishment commands support the -s flag for silent execution (hidden from public chat).

Command Permission Description
/ban <player> [reason] controlbans.ban Permanently bans a player.
/tempban <player> <time> controlbans.tempban Temporarily bans a player (e.g. 1d12h).
/mute <player> [reason] controlbans.mute Permanently mutes a player from chat.
/tempmute <player> <time> controlbans.tempmute Temporarily mutes a player.
/voicemute <player> controlbans.voicemute Mutes a player in SimpleVoiceChat only.
/voidjail <player> controlbans.voidjail Traps player in an inescapable void box.
/banskin <player> controlbans.banskin Forcibly removes a player's skin.
/kick <player> controlbans.kick Disconnects the player from the server.

Utilities & Tools

Command Permission Description
/history <player> controlbans.history Opens a GUI showing past punishments.
/alts <player> controlbans.alts Scans for accounts sharing the same IP.
/freeze <player> controlbans.freeze Freezes player movement for inspection.
/check <player> controlbans.check Checks active punishment status.
