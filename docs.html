Installation

  1. Download the plugin from BuiltByBit or Modrinth.
  2. Place the jar in your /plugins folder.
  3. Restart your server to generate configuration files.
  4. (Optional) Install DiscordSRV for Discord logs and SimpleVoiceChat for AI Voice Moderation.
Network Setup (Bungee/Velocity)

Install ControlBans-Bridge.jar on your proxy and the main plugin on all backend servers for cross-server sync.

Migration

Migrate from other plugins with a single command. Ensure both plugins are installed.

/controlbans import <litebans|essentials|advancedban|file>
Pro Tip: Add -s to the end of any punishment command to execute it silently (e.g. /ban Notch -s).

Main Configuration

Key settings from config.yml.

config.yml
database:
  # Options: mysql, mariadb, postgresql, sqlite, h2, mongodb
  type: sqlite
  
redis:
  enabled: false # Set true for cross-server sync
  host: localhost
  port: 6379

web:
  enabled: true
  port: 8080
  admin-token: "change-me"

integrations:
  discord:
    enabled: true
    # Send embeds via DiscordSRV
    channel-id: "123456789"
    
void-jail:
  location:
    world: "world"
    x: 0.5
    y: -1000.0 # Far below the world
    z: 0.5

AI & Voice Config

Settings for AI moderation in ai-config.yml.

ai-moderation:
  enabled: true
  provider: "groq"
  api-key: "insert_your_key_here"
  model: "llama3-8b-8192"
  
  actions:
    # Command to run when AI detects toxicity
    violation-command: "tempban {player} 7d [AI] {reason} (Conf: {confidence})"
    min-confidence: 0.85 # Only punish if 85% sure

voice-moderation:
  enabled: true
  model: "whisper-large-v3"
  silence-timeout: 1.5
