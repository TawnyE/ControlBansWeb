Installation

  1. Download the plugin from BuiltByBit or Modrinth.
  2. Place the jar in your /plugins folder.
  3. Restart your server to generate configuration files.
  4. (Optional) Install DiscordSRV for Discord logs and SimpleVoiceChat for AI Voice Moderation.
Network Setup (Bungee/Velocity)

Install ControlBans-Bridge.jar on your proxy and the main plugin on all backend servers for cross-server sync.

Migration

Migrate from other plugins with a single command. Ensure both plugins are installed.

/controlbans import <litebans|essentials|advancedban>

Commands

Add -s to any punishment command to execute it silently.

Command Permission Description
/ban <player> [reason] controlbans.ban Permanently ban a player.
/tempban <player> <time> controlbans.tempban Temporarily ban (e.g. 1d).
/mute <player> [reason] controlbans.mute Permanently mute chat.
/voicemute <player> controlbans.voicemute Mute in Voice Chat only.
/kick <player> controlbans.kick Kick a player.
/warn <player> [reason] controlbans.warn Issue a warning.
/voidjail <player> controlbans.voidjail Trap player in void box.
/banskin <player> controlbans.banskin Remove player's skin.
/freeze <player> controlbans.freeze Freeze for inspection.
/history <player> controlbans.history View punishment history GUI.
/alts <player> controlbans.alts View linked accounts (IP).
/chat <lock|slow|clear> controlbans.chat Manage global chat.

Extra Permissions

Node Description
controlbans.adminFull access to reload, import, and settings.
controlbans.notify.silentSee silent punishment broadcasts.
controlbans.exempt.*Exempt from bans/mutes/kicks.
controlbans.bypass.chatTalk when chat is locked/slowed.

Main Configuration

Key settings from config.yml.

database:
  # Options: mysql, mariadb, postgresql, sqlite, h2, mongodb
  type: sqlite
  
redis:
  enabled: false # Set true for cross-server sync
  host: localhost
  port: 6379

web:
  enabled: true
  port: 8080
  admin-token: "change-me"

integrations:
  discord:
    enabled: true
    # Send embeds via DiscordSRV
    channel-id: "123456789"
    
void-jail:
  location:
    world: "world"
    x: 0.5
    y: -1000.0 # Far below the world
    z: 0.5
