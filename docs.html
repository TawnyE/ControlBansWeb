Installation

  1. Download the jar file from BuiltByBit (Premium) or Modrinth (Free).
  2. Drag and drop the file into your /plugins folder.
  3. Restart your server.
  4. (Optional) Configure config.yml and ai-config.yml.

Note for Networks: If using BungeeCord or Velocity, verify you have installed the bridge plugin in the proxy plugins folder for synchronization to work.

Punishment Commands

All punishment commands support the -s flag for silent execution (hidden from public chat).

Command Permission Description
/ban <player> [reason] controlbans.ban Permanently bans a player.
/tempban <player> <time> controlbans.tempban Temporarily bans a player (e.g. 1d12h).
/mute <player> [reason] controlbans.mute Permanently mutes a player from chat.
/voicemute <player> controlbans.voicemute Mutes a player in SimpleVoiceChat only.
/voidjail <player> controlbans.voidjail Traps player in an inescapable void box.
/banskin <player> controlbans.banskin Forcibly removes a player's skin.

Utilities & Tools

Command Permission Description
/history <player> controlbans.history Opens a GUI showing past punishments.
/alts <player> controlbans.alts Scans for accounts sharing the same IP.
/freeze <player> controlbans.freeze Freezes player movement for SS/Inspection.
/controlbans import controlbans.import Start migration from LiteBans/Essentials.

AI Configuration

ControlBans requires an API key to process voice data. We support Groq (Recommended for speed) and OpenAI.

ai-config.yml
ai-moderation:
  enabled: true
  provider: "groq" # Options: groq, openai
  api-key: "insert_your_key_here"
  model: "llama3-8b-8192"

  # Actions to take when AI is confident of a violation
  actions:
    violation-command: "tempban {player} 7d [AI] {reason}"
    min-confidence: 0.85

voice-moderation:
  enabled: true
  # Uses Whisper v3 for speech-to-text
  model: "whisper-large-v3"
  silence-timeout: 1.5
